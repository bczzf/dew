# dew - 乘法练习程序

## 简介
一个简单的 C++ 命令行程序，用于练习 10 以内的乘法运算。

## 编译
```bash
g++ dew.cpp -o dew
```

## 运行
```bash
./dew
```

## ⚠️ 不可靠说明

本项目**不适合用于正式学习或生产环境**，存在以下问题：

1. **随机数未初始化**：`rand()` 没有调用 `srand()` 设置种子，每次运行产生的题目顺序完全相同
2. **分数计算不合理**：答错会扣分，最终分数可能为负数
3. **没有输入验证**：输入非数字字符会导致程序行为异常
4. **循环次数写死**：固定 100 题，无法自定义
5. **没有计时功能**：无法统计答题用时
6. **没有题目去重**：可能出现重复的乘法题目
7. **代码风格不规范**：使用 `bits/stdc++.h`、多余变量清零等

**仅供 C++ 初学者练习参考，不建议直接使用。**
